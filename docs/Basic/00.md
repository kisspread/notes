  - PSO 是 Pipeline State Object (管线状态对象) 的缩写, 当 UE5 遇到一个新的材质组合时，需要编译对应的 PSO 并提交给 GPU。这个编译过程发生在运行时，就会造成帧率下降和卡顿。
